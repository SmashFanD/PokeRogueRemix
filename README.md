PokeRogueRemix is a small project that i'm making, using PokeRogue as a base of interest from wich i will use some assets. (code will NOT be copy/pasted)

What will the project contain ?

 -Not as much content.
 -Starter Selection.
 -Only Solo battle Mode.
 -Turn by Turn Mystery Dungeon style (PKMN1 Choose Action -> Do Action -> PKMN2 Choose Action -> Do Action).
 -No Pokeball, switching does not reset any status or stats change, wild pokemon decide themself if they want to join.
 -DayCare system, you can choose to send pokemon to the daycare, they will gain lvl, and you can choose to use them again, this will heal them a little.
 -Move with cooldown instead of pp, with strong moves (like healing) having their cooldown increased after each use.
 -Number of attacks depend on the pokemon speed at the start of the turn, Actions=PKMNSpeed/SlowestPKMN, with a cap of 3-4 depending of linked move.
 -When using an attack you can't use item, switch or run this turn.
 -Linked Moves, with an item you will be able to link two moves, using both moves in one action but changing adding their cooldown.
 -Move Lvl up, increasing accuracy or Power depending on if it's a status move or attck move, with some exceptions (Start at lvl0, +2power/accuracy each lvl)
 -Lvl from moves are stored in the run and are shared for each team member.
 -Accuracy and Crit rate based on speed difference, no evasion stat and Moves can have over 100% accuracy (120% almost always hit, 125% to always hit).
 -Freeze will be a paralysis that Ice types are immune to, lowering speed (no longer make the move fail).
 -Sleep treat speed as 0, garanting crit and moves to hit the sleeping target.
 -PKMN can sometimes heal status when they are not on field, same for stats changes one at a time.
 -PKMN can have multiples non-volatiles Status.
 -Weak badge boost will be in, with Champions Ribons giving a bigger boost.
 -Pokerus (to not forget), evs will work kinda the same as gen 1 (But need 60000pt to get 60evs for each stats at max).
 -Boss each 5 wave instead of 10, faster progression.
 -Bag to store items.
 -items (only usable, or givable in battle, no out of battle item).
 -Shop.
 -Tokens.
 -Saves.
